● What is SPA?
Single page application is an application that loads one HTML page,
It dynamically update the page without reloading to load other components.

● What is difference between Client Side Routing and Server Side Routing?
Client side routing is done for navigation purpose to travel into different component.
Page do not reload when we navigate to other pages.
use browser APIs.

Server side routing is like reloading the page everytime. server returns new html
